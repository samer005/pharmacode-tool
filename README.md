# Pharmacode Reader & Generator

A mobile-friendly Progressive Web App (PWA) for generating, decoding, and validating pharmaceutical Pharmacodes (Laetus Pharmacode).

Created by Samer M.

---

## Features

### Generate Pharmacodes

* Enter a Pharmacode number
* Generate the corresponding barcode
* Supports Pharmacode values within the supported range
* True narrow/wide bar representation

### Decode Pattern

Decode directly from a bar pattern:

Examples:

NNNNWNWWNNN

or

N N N N W N W W N N N

Where:

* N = Narrow Bar
* W = Wide Bar

### Photo Reader

* Upload an image from your device
* Decode Pharmacodes from photos
* Mobile camera support
* Designed for iPhone and Android devices

### Crop Scanner

* Select only the barcode area
* Improves decoding accuracy
* Useful for packaging photos and labels

### Export Options

* PNG
* SVG
* PDF
* EPS

### Progressive Web App (PWA)

* Installable on iPhone and Android
* Full-screen experience
* Home screen icon
* Offline support

## Installation

### GitHub Pages

1. Upload all files to your repository root.
2. Go to:

Settings → Pages

3. Configure:

Source: Deploy from a branch

Branch: main

Folder: /(root)

4. Save.

GitHub will publish your website automatically.

---

## iPhone Installation

1. Open the GitHub Pages URL in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Launch the application from your home screen.

The application will behave similarly to a native app.

---

## Recommended Usage

### Best Results When Reading Photos

For highest accuracy:

* Use good lighting
* Avoid shadows
* Keep the barcode horizontal
* Fill most of the image with the Pharmacode
* Use the crop tool if the barcode occupies a small area
