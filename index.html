Decode Pattern
Type N for narrow and W for wide.
Range note: This tool can generate 1–131070 as requested. In common one-track Pharmacode validation, 1–2 are test-only single-bar outputs; industrially documented Pharmacode normally starts at 3.
Standard dimensions (Laetus spec):
Thin bar (a1) = 0.5 mm
Thick bar (b1) = 1.5 mm  → ratio 3:1
Gap (c1) = 1.0 mm
Height (e1) = 8.0 mm minimum
Colors: black bars, white background
